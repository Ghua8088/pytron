# Security Policy

## Supported Versions

We only support security updates for the latest major version of Pytron.

## Security Changes

We have recently added packaging with Secure ,Nuitka pipeline and they are the only safest way to package Pytron applications, *the default PyInstaller is not safe and can be exploited.*

## Reporting a Vulnerability

If you discover a security vulnerability within Pytron, please send an e-mail to the maintainer (Ghua8088) or open a security advisory on GitHub.

Please include:
- A description of the vulnerability.
- Steps to reproduce (or a proof-of-concept script).
- Potential impact.

We will acknowledge your report and keep you updated on the progress of the fix. Please do not disclose the vulnerability publicly until a fix has been released.
