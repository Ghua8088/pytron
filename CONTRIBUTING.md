# Contributing to Pytron

Thank you for your interest in contributing to Pytron! We welcome contributions from everyone.

## How to Contribute

1.  **Report Bugs**: Use GitHub Issues to report bugs. Please include steps to reproduce the bug and your environment details.
2.  **Suggest Features**: Open a GitHub Issue to suggest new features or improvements.
3.  **Submit Pull Requests**: 
    *   Fork the repository.
    *   Create a new branch for your feature or bug fix.
    *   Ensure your code follows the existing style.
    *   Write or update tests if necessary.
    *   Submit a PR with a clear description of your changes.

## Development Setup

1.  Clone the repository:
    ```bash
    git clone https://github.com/Ghua8088/pytron.git
    cd pytron
    ```

2.  Install dependencies:
    ```bash
    pip install -r requirements.txt
    ```

3.  Run tests:
    ```bash
    pytest
    ```

## Code of Conduct

Please be respectful and professional in all interactions within the Pytron community.
